基于 GatewayWorker 框架的在线客服系统 
=================
本项目是一个在线客服系统的demo,主要通过gatewayworker框架创建websocket和webserver等服务实现一对一和一对多客服功能。

启动
=======
双击start_for_win.bat


当前版本的功能
============

1. 客服账号密码登录
2. 在线客服数量自动显示，并实时更新，方便访客找到最合适的客服
3. 支持客服一对多管理

未来计划实现的功能
===============
1. 更加美观的界面
2. 与lareval,Tp5框架进行结合
3. 保存聊天记录
4. 进行访客信息统计
